# 0.6 release notes

_In development._ Changes land here as they're committed; this file becomes the
GitHub release body when `v0.6` is tagged (release.yml reads it by version).

## New features

- **BAIN archive installer.** Mods packaged the Wrye Bash way — content split
  into numbered folders like `00 Core`, `01 Optional Textures`, `10 Alternate
  Meshes` — now get a package picker at install time instead of installing as
  one blob. Tick the parts you want (everything is selected by default) and the
  chosen packages are merged into the mod. FOMOD installers still take
  precedence; this only kicks in for the numbered-folder layout.

## Bug fixes

- **Flatpak LOOT now works.** When LOOT is installed as a Flatpak, "Sort with
  LOOT" used to run it but it detected no plugins and closed — the Flatpak
  sandbox couldn't see your mods folders or the OpenMW config. The manager now
  launches Flatpak LOOT through `flatpak run` with the filesystem access it
  needs (mods/plugins on any drive, plus read/write to `openmw.cfg`), so sorting
  works the same as a native LOOT install. Native installs are unaffected.
